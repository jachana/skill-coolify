# coolify

Manage a self-hosted Coolify PaaS over its API — list/inspect/create/update/delete applications, databases, services, servers, projects, and environment variables

## Install (Claude Code)

```
/plugin marketplace add jachana/claude-marketplace
/plugin install coolify@jachana
```

Part of the [jachana marketplace](https://github.com/jachana/claude-marketplace).
